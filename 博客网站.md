# 博客网站自主设计

###  一.需求分析

1. 形成属于自己的编程知识库
2. 显化知识

### 二.功能设计

1. 文章功能的增删改查，支持上传和解析各类型文件（png,html,md,pdf等等）,通过大模型实现对文章标题，问题描述，经验总结的自动填写。
2. 大模型对所有经验总结进行归类总结并存储放入经验知识库并进行分类，按我打开的的频率和默认排序（由AI根据价值打分），并支持搜索，当发现相似度过高的经验，不加入知识库。
3. 设计一个Agent智能体，实现通过引导对话，完成文章高质量自动发布。

### 三.技术选型

1. 前端：Vite + React+ Tailwind CSS + Axios
2. 后端：SpringBoot3+SpringAI+MySQL+ElasticSearch+Kafka+Minio

### 四.表设计

**三件套** ：creattime updatetime is_delete

2. empirical knowledge（经验知识表）

(
    id             int auto_increment
        primary key,
    article_id_emp int                                null comment '外键关联文章id',
    type           int                                not null comment '类型',
    frequency      int      default 0                 not null comment '频率',

    constraint article_id_emp
        foreign key (article_id_emp) references article (id)
